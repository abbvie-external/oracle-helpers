BindWithName

BindWithName: BindParameter & { name: string }
  • Defined in lib/sql.ts:22

Provides the ability to Bind a parameter with a name that will be used

This is very useful for the output values

example
const title = 'Good Omens';
const author = 'Terry Pratchett & Neil Gaiman';
const pages = 288;
const bind = {
  name: 'id',
  dir: BIND_OUT,
  type: NUMBER,
};
const query = sql`INSERT INTO books (TITLE, AUTHOR, PAGES)
                             VALUES (${title}, ${author}, ${pages})
                  RETURNING ID into (${bind})`;

getPoolConnection

  • getPoolConnection(dbConfig: ConnectionAttributes): Promise<Connection>
    • Defined in lib/pools.ts:90

    Gets a connection from a pool. Will run createPool automatically

    Has a 3 seconds timeout before it abandons the currently extant pool and tries again. This is in order to make the application able to recover.

    Will also run a ping on a connection at least every configuration.pingTime miliseconds. This is set to 1 minute by default as it can be slower.

    Parameters

    • dbConfig: ConnectionAttributes

    Returns Promise<Connection>

    an oracle connection object
