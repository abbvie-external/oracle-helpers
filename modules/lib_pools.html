getPoolConnection

  • getPoolConnection(dbConfig: ConnectionAttributes): Promise<Connection>
    • Defined in lib/pools.ts:86

    Gets a connection from a pool. Will run createPool automatically

    Has a 3 seconds timeout before it abandons the currently extant pool and tries again. This is in order to make the application able to recover.

    Will also run a ping on a connection at least every configuration.pingTime miliseconds. This is set to 1 minute by default as it can be slower.

    • dbConfig: ConnectionAttributes
